Explicación

Los programadores de C# usan un bloque try para separar el código que podría verse afectado por una excepción. Los bloques catch asociados se usan para controlar las excepciones resultantes. Los bloques finally contienen código que se ejecuta independientemente de si se produce una excepción en el bloque try, como la liberación de recursos asignados en el bloque try. Los bloques try requieren uno o varios bloques catch asociados, un bloque finally o ambos.
