Explicación


Definición del método en C#: DateTime TryParse(String, IFormatProvider, DateTimeStyles, DateTime)

Description

DateTime TryParse(String, IFormatProvider, DateTimeStyles, DateTime)

convierte la representación de cadena especificada de una fecha y hora en su equivalente DateTime, utilizando la información de formato específico de la cultura y el estilo de formato especificado, devolviendo un valor falso o verdadero que indica si la conversión se realizó correctamente.

Síntaxis

static bool TryParse (
string s, // ' 19/11/2020 10:00 AM '
IFormatProvider provider,
DateTimeStyles styles,
out DateTime result
)

Parámetros

DateTime.TryParse(String, IFormatProvider, DateTimeStyles, DateTime)

Definiendo los parámetros descritos en el método:

s - Una cadena que contiene una fecha y hora para convertir.

provider - Un objeto que proporciona información de formato específica de la cultura sobre 's'.

style - Una combinación bit a bit de valores de enumeración que define cómo interpretar la fecha analizada en relación con la zona horaria actual o la fecha actual. Un valor típico para especificar es Ninguno.

result - Cuando este método regresa, contiene el valor DateTime equivalente a la fecha y hora contenidas en 's', si la conversión tuvo éxito, o MinValue si la conversión falló. La conversión falla si el parámetro 's' es nulo, es una cadena vacía ("") o no contiene una representación de cadena válida de una fecha y hora. Este parametro se pasa sin inicializar.

Regresa

DateTime.TryParse(String, IFormatProvider, DateTimeStyles, DateTime)
El método devuelve verdadero si el parámetro 's' se convirtió correctamente; de lo contrario, falso.
